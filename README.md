#  About COSMOS

COSMOS Explorer is an universe exploration platform designed to transform the way users experience space on the web.

Instead of simply displaying information, COSMOS creates a cinematic and immersive digital universe where users can:

- Explore planets in 3D
- Experience galaxy and black hole simulations
- Track the International Space Station live
- View NASA’s Astronomy Picture of the Day
- Learn through quizzes and timelines
- Enjoy futuristic visuals, animations, and ambient soundscapes

The project combines creativity, science-inspired visuals, and modern frontend technologies into a smooth browser-based experience.

#  Features

##  Interactive Solar System
Explore animated planets with orbit paths, atmospheric glows, and immersive visuals.

---

##  Galaxy Explorer
Travel through beautifully generated galaxies inspired by:
- Milky Way
- Andromeda
- Whirlpool Galaxy
- Sombrero Galaxy

Built using custom particle systems and mathematical distributions.

---

##  Black Hole Simulation
A cinematic black hole inspired by *Interstellar*, featuring:
- Gravitational lensing effects
- Photon rings
- Dynamic event horizon rendering
- Interactive viewport distortion

---

##  Real-time ISS Tracker
Track the live movement and coordinates of the International Space Station as it orbits Earth.

---

##  NASA APOD Integration
Displays NASA’s Astronomy Picture of the Day directly using NASA Open APIs.

---

##  Space Quiz & Timeline
Interactive educational modules designed to make space exploration more engaging and fun.

---

##  Immersive Experience
Ambient sound effects and smooth cinematic animations create the feeling of being inside a futuristic spaceship dashboard.

# ⚙️ Installation & Setup

Follow these steps to run COSMOS locally on your machine.

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/sreyanandi/COSMOS.git
```

## 2️⃣ Navigate to the Project Folder

```bash
cd COSMOS
```

## 3️⃣ Install Root Dependencies

```bash
npm install
```

## 4️⃣ Install Frontend Dependencies

```bash
cd frontend
npm install
```

## 5️⃣ Install Backend Dependencies

```bash
cd ../backend
npm install
```

---

# ▶️ Running the Project

##  Start Frontend

Open a terminal window:

```bash
cd frontend
npm run dev
```

Frontend will run on:
🌐 [http://localhost:5173](http://localhost:5173)

##  Start Backend

Open another terminal window:

```bash
cd backend
npm run dev
```

Backend will run on:
🌐 [http://localhost:3001](http://localhost:3001)
